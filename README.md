## Описание
Простой, быстрый и красивый кроссплатформенный калькулятор

## Особенности
* Кроссплатформенность
* Хороший дизайн
* Быстрый запуск
* Легкий в плане веса и потребления ОЗУ

## Системные требования
* **Операционная система:** Windows XP или выше/Linux
* **Минимальное количество оперативной памяти:** 30 МБ
* **Рекомендуемое количество оперативной памяти:** ~100 МБ
* **Дополнительно:** Java 8 (для ревизии *JCalcX_Windows_With_JRE* - необязательно, только Windows) 

## Сборка (требуется DevelNext и JDK)
1. Скачиваем репозиторий
2. Скачиваем и устанавливаем [DevelNext](https://github.com/jphp-group/develnext/releases) и [JDK](https://www.oracle.com/technetwork/java/javase/downloads/2133151)
3. Открываем скачанную версию калькулятора через "открытие проектов" в DevelNext
4. Собираем проект 
5. Выбираем "Открыть папку", копируем полученный файл и закрываем DevelNext
6. Запускаем и радуемся!
